 1e15를 입력하면 과학적 표기법으로 표시하지 않고, 
```python
print(1e15)
# 1000000000000000.0
```
 
1e16을 입력하면 과학적 표기법으로 표시된다. 
``` python
print(1e16)
# 1e+16
```
또한 1e-4를 입력하면 과학적 표기법으로 표시하지 않고, 
```python
print(1e-4)
# 0.0001
```
1e-5를 입력하면 과학적 표기법으로 표시한다.
```python
print(1e-5)
# 1e-05
```
