# 함수와 메소드의 차이
```css
# 함수(function)

프로그래밍 언어에서 function(함수)는 어떠한 기능을 수행하는 도구이다.
각 함수는 각각 다른 기능을 가지고 있다.
ex) sum(): 총 합, max(): 최댓값 , abs(): 절대값,.....
이와 같이 프로그래밍 세계에는 수많은 함수가 있고, 사용자가 직접 만든 함수를 만들 수도 있기 때문에 
수를 한정할 수 없다.

# 메소드(method)

메소드(method)도 함수다.
따라서, 메소드도 함수처럼 어떤 기능을 수행하지만, 메소드는 " 객체와 클래스 " 에 연관이 있는 함수이다.
object(객체)와 연관되어 사용된다. → 사용하고자 하는 대상이 "." 으로 연결되어 있어야 한다.

ex) a.upper() : 객체 a를 대문자로 바꾼다 , a.pop(): 리스트의 가장 뒤쪽에 있는 데이터(원소)를 빼낸다. , a.append(): 리스트 가장 뒤에 데이터를 추가한다.

# 결론

함수: 독립적으로 정의되므로 이름으로만 호출이 가능함.
메소드: 이름으로만 호출되지 않음. 정의된 클래스의 참조에 의해 클래스를 호출해야한다. 메소드는 클래스 내에서 정의되므로 해당 클래스에 종속된다.
```
