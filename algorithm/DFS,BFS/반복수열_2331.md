# 문제: 반복수열_2331
https://www.acmicpc.net/problem/2331

그냥 구현 문제 인것 같다... 왜 dfs?
# 풀이

``` python
import sys
sys.setrecursionlimit(10**6)
input =lambda : sys.stdin.readline().rstrip()


a,p = map(int,input().split())
# 반복 수열 식
arr= [a]

while True:

    a = str(a)
    tmt = 0
    for i in a:
        tmt += (int(i) ** p)

    if tmt in arr:
        for i,k in enumerate(arr):
            if k == tmt:
                print(i)
                break
        break
    arr.append(tmt)
    a = tmt
```
