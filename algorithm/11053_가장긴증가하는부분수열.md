# 11053_가장긴증가하는부분수열

## 입력값
## 첫 줄: 6
## 두번째 줄: 10 20 10 30 20 50


# 방법 1
``` python
n = int(input())
a = [0]+ list(map(int, input().split())) # a는 수열
d = [0]+[1] * n # d는 a[i]를 가장 큰 수로 갖는 가장 긴 증가하는 부분수열

for i in range(2,n+1):
    for j in range(1,i):
        if a[i] > a[j]:
            if d[i] <= d[j]:
                d[i] = d[j] + 1


print(max(d))
``` python
# 방법 2
n = int(input())
a = [0] + list(map(int, input().split()))
d = [0] * (n+1)
d[1] = 1
for i in range(1,n+1):
    s= []
    for j in range(0,i):
        if a[i] > a[j]:
            s.append(d[j]+1)

    d[i] = max(s)


print(max(d))
```
