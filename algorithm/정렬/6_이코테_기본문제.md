입력값:
5 3
1 2 5 4 3
5 5 6 6 5

출력값:
26

# 1번 풀이
파이썬 기본 정렬 라이브러리 사용

``` python
n, k = map(int, input().split())

a = list(map(int, input().split()))
# 1 2 5 4 3
b = list(map(int, input().split()))
# 5 5 6 6 5

# a 정렬, b정렬
a.sort() # [1,2,3,4,5]
b.sort()
b.reverse()# [6,6,5,5,5]

tmp = k
for i in range(n):# 3번까지 교체
    if tmp == 0:
        break
    if a[i] < b[i]:
        a[i] = b[i]
        tmp -= 1

ans = 0
for i in a:
    ans += i
print(ans) # 출력값: 26



```
