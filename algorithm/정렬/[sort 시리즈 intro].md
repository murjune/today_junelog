# 정렬 알고리즘 종류와 특징
```
# 구현은 쉽지만 실행시간이 긴 것 : 

1. Selection Sort(선택 정렬)
2. Bubble Sort (버블 정렬) 

시간복잡도 : O(N^2)

# 구현은 어렵지만 실행시간이 짧은 것 :

3. Merge Sort(머지 정렬) 
4. Quick Sort 
5. Time Sort 

시간복잡도 : O(N * logN)
```

# Stable sort란?

```
Stable sort: 정렬 후 기존의 순서가 유지되는 정렬
머지 정렬, 버블 정렬, 삽입 정렬은 Stable 하다.
```
