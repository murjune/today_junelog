# 문제: k번째수_11004
https://www.acmicpc.net/problem/11004

# 풀이 1
``` python
import  sys
read = lambda :sys.stdin.readline().rstrip()
write = lambda x: sys.stdout.write((str(x)+ "\n"))
# 입력

n, k = map(int, read().split())

arr = list(map(int, read().split()))

# 정렬(파이썬 기본 라이브러리)
arr.sort()

write(arr[k-1])
```
