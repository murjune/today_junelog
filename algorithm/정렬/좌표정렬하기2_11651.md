문제: 좌표 정렬하기2_11651
https://www.acmicpc.net/problem/11651

# 방법1
``` python


# 입력
n = int(input())
arr =[[0,0]for _ in range(n)]

for i in range(n):
    arr[i] = list(map(int, input().split()))
# arr = [[3, 4], [1, 1], [1, -1], [2, 2], [3, 3]]

# reverse -> 정렬
for i in arr:
    i.reverse()

arr.sort()

for i in arr:
    i.reverse()

# 출력
for i in arr:
    result = ' '.join(map(str, i))
    print(result)


```
