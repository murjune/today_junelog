# 2011: 암호코드

https://www.acmicpc.net/problem/2011

```
(문제풀이)
경우의 수를 나누어 점화식을 구성하였다.

경우1) a[n]과 a[n-1]이 독립적인 암호를 갖을 때 25 -> BE

경우1을 만족 하기 위해서는 
1 <= a[i] <= 9를 만족해야한다. 
d[i] += d[i-1]

경우2) a[n]a[n-1]가 하나의 암호를 갖을 때 12 -> Y

경우2를 만족시키 위해서는
10 <= a[i-1] * 10 + a[i] <= 26를 만족해야한다. 

d[i] += d[i-2] (이때, i != 1 이여야 한다.)

경우3)
이 때 경우 1과 경우 2를 만족하지 않으면 0을 출력하게 코딩한다.

위의 경우를 모두 만족하는 코드를 짠다.
```

``` python
a = []
for i in list(n):
    a.append(int(i))
# 다음과 같이 한줄로 나타낼 수 있다.

a = [int(i) for i in list(n)]
```


``` python
n = input()  # '25114'
a = [0]+[int(i) for i in list(n)]  # a =[0,2,5,1,1,4]
m = len(a)-1 # 5
d = [0] * (m+1) # d =[1,0,0,0,0,0]
if a[1] !=0:
    d[0] =1
    for i in range(1, m+1):
        # (경우 1)
        if 1 <= a[i] <= 9: # 1 <= a[n-1] <= 9
            d[i] += d[i-1]
        # (경우 2)
        if i == 1:
            continue
        if 10 <= a[i-1] * 10 + a[i] <= 26:
            d[i] += d[i-2]

print(d[m] % 1000000)


```
