# 최소공배수

## 파이썬은 math 모듈을 사용하여 구할 수 있다.
``` python
import math

a, b = map(int, input().split())

print(math.lcm(a, b))
```
```
최소공배수 (LCM)

L * g = a * b ->  L = a * b / g
L = g * (a/g) * (b/g)

int 값의 범위 : 	(–2,147,483,648 ~ 2,147,483,647)
a , b가 int 값의 범위에 들어올지라도, L값이 int 값의 범위를 넘어갈 수 있다.
따라서, 범위를 체크하여 올바른 자료형을 사용하는  것이 중요하다.
```
