# 문제: 2진수 8진수_1373

https://www.acmicpc.net/problem/1373

## 실패 (시간초과)
``` python
import sys

n = (sys.stdin.readline().strip()) # 11001100


tmp = 0  # ord(1) = 49 , ord(0) =48

# 2진법 -> 10진법
for i in n:
    tmp *= 2
    tmp += (ord(i)-48)

# tmp: 204 (n -> 10)

answer = ''

while tmp / 8 != 0:

    answer += str(tmp % 8)
    tmp //= 8


#answer = '413'

print(answer[::-1])
```
아마 n의 길이가 너무 길게 되면,  while 반복문에서 처리수가 급증하여 시간복잡도가 O(N)일지라도 시간초과가 일어나는 것 같다. 

## 방법 1

``` python

a = input() # 11001100

if len(a) % 3 == 1: 
    a = '00'+ a # 문자열 길이를 3의 배수로 만들어 주기 위해

elif len(a) % 3 == 2: # 011 001 100
    a = '0' + a # 문자열 길이를 3의 배수로 만들어 주기 위해


n = len(a) # 9

tmp = 0
oct_list = []

for i in range(0 , n): # n: 9 , i : 0~8
    tmp *= 2
    tmp +=  (ord(a[i])- 48) # ord(1) = 49 , ord(0) =48

    if i % 3 == 2: 
        oct_list.append(tmp)
        tmp = 0

answer = ''
for i in oct_list:
    answer += str(i)

print(answer)


```
