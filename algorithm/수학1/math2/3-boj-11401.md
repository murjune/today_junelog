# [이항계수 3](https://www.acmicpc.net/problem/11401)

이 문제는 N의 범위가 너무 크기 때문에 O(n<sup>2</sup>)의 알고리즘으로는 풀 수 없는 문제이다.  
먼저, 분수의 경우 모듈러 연산을 적용할 수 없다. 분모,
