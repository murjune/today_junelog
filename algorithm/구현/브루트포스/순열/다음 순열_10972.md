# 문제: 다음 순열
https://www.acmicpc.net/problem/10972. 


순열을 사전순으로 나열했을 때, 다음에 오는 순열을 찾는 문제  

# 풀이
```
# 다음 순열 찾는 방법
1. A[i-1] < A[i]를 만족하는 가장 큰 i를 찾는다. (i는 인덱스)
2. j >= i, A[j] > A[i-1]를 만족하는 가장 큰 j를 찾는다.
3. A[i-1], A[j]를 Swap
4. i번째 수부터 끝까지 정렬
```
``` python

# 1. A[i-1] < A[i] 만족하는것중 가장 큰 i찾기
def I():
    tmp = -100
    for i in range(1,n):
        if A[i-1] < A[i]:
            tmp = max(tmp,i)
    return tmp
# 2. j >= i 이면서 A[j] > A[i-1] 만족하는 가장 큰 j찾기

def J(tmp):
    tmp2 = 0
    for j in range(tmp,n):
        if A[tmp-1] < A[j]:
            tmp2 = max(tmp, j)
    return tmp2
#3. A[tmp-1], A[tmp2] swap
#4. i 부터 배열 뒤집기
def swap(tmp,tmp2):
    global A
    A[tmp-1], A[tmp2] = A[tmp2], A[tmp-1]
    b = A[tmp:]
    A = A[:tmp] + b[::-1]


n = int(input()) # N(1 ≤ N ≤ 10,000)
A = list(map(int, input().split()))
i = I()
if i < 1:
    print(-1)
else:
    j = J(i)
    swap(i,j)
    print(' '.join(map(str, A)))
```
