# Heap

힙은 일종의 Complete Binary Tree이다.  
힙은 크게 Max_Heap과 Min_Heap으로 나뉜다.
```
최대 힙: 부모 노드 값 > 자식 노드 값
최소 힙: 부모 노드 값 < 자식 노드 값
```
